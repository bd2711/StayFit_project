# StayFit - Personalized AI Health Coach (Mentorship Project)

StayFit is a web application designed to help users manage their health goals through AI-powered coaching and data-driven insights. This project is part of my mentorship program, and I will be building it out over the next few weeks.

## Week 1: Project Foundation
The focus for this week was setting up the project structure and development environment:
- **Backend Setup**: Initialized FastAPI structure and dependencies.
- **Frontend Setup**: Initialized React/Vite structure with TailwindCSS.
- **Configuration**: Set up environment variable management and git version control.

## Project Structure
- `backend/`: Python FastAPI application logic.
- `frontend/`: React + Vite frontend application.

## How to Set Up (Dev Environment)

### 1. Prerequisites
- Python 3.10+
- Node.js & npm

### 2. Backend Installation
```bash
cd backend
pip install -r requirements.txt
```

### 3. Frontend Installation
```bash
cd frontend
npm install
```
